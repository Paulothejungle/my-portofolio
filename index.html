Tentang Saya

Saya adalah seorang backend developer dengan pengalaman dalam merancang, mengembangkan, dan memelihara sisi server dari aplikasi web. Keahlian utama saya terletak pada pembuatan RESTful API dan manajemen database. Saya selalu antusias untuk belajar teknologi baru dan menerapkannya untuk memecahkan masalah yang kompleks secara efisien.

Di luar coding, saya suka bermain game dan jogging yang membantu saya berpikir kreatif dan menjaga keseimbangan hidup.
